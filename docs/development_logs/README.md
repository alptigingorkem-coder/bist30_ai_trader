# Geliştirme Günlükleri

Bu klasör, projede yapılan geliştirmelerin günlük bazında kaydını tutar.

## Dosya Formatı

Her dosya `YYYY-MM-DD.md` formatında adlandırılır.

## Mevcut Günlükler

| Tarih | Özet |
| :--- | :--- |
| [2026-02-03](2026-02-03.md) | Trade Ledger, Confidence Analizi, Live Stress, Live Trading |

## Şablon

Yeni günlük oluştururken:

```markdown
# Geliştirme Günlüğü - YYYY-MM-DD

## Yapılanlar
- ...

## Sonuçlar
- ...

## Notlar
- ...
```
