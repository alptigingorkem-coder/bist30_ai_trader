# Katkı Sağlama Rehberi (Contributing Guide)

BIST30 AI Trader projesine katkıda bulunmak istediğiniz için teşekkür ederiz! Açık kaynak topluluğunun gücüne inanıyoruz ve her türlü katkıyı (kod, dokümantasyon, hata bildirimi) memnuniyetle karşılıyoruz.

## Nasıl Katkıda Bulunabilirsiniz?

### 1. Hata Bildirimi (Bug Reporting)
Karşılaştığınız hataları veya sorunları GitHub "Issues" sekmesi üzerinden bildirebilirsiniz.
Lütfen bildirimde bulunurken şunları ekleyin:
- Hatanın kısa bir tanımı.
- Hatayı tekrar etmek için gerekli adımlar.
- Beklenen ve gerçekleşen sonuç.
- Varsa hata mesajları veya ekran görüntüleri.
- İşletim sistemi ve Python sürümünüz.

### 2. Yeni Özellik Önerisi (Feature Request)
Projeye eklenmesini istediğiniz özellikleri yine "Issues" sekmesi üzerinden "Enhancement" etiketiyle belirtebilirsiniz.

### 3. Kod Katkısı (Pull Requests)
1.  Projeyi kendi hesabınıza **Fork** edin.
2.  Kendi bilgisayarınıza klonlayın: `git clone https://github.com/SizinKullaniciAdiniz/bist30_ai_trader.git`
3.  Yeni bir **Branch** oluşturun: `git checkout -b feature/muhtesem-ozellik`
4.  Geliştirmelerinizi yapın ve **Commit** edin: `git commit -m 'feat: Muhteşem özellik eklendi'`
5.  Branch'inizi **Push** edin: `git push origin feature/muhtesem-ozellik`
6.  GitHub üzerinden **Pull Request (PR)** açın.

### Kod Standartları
- Python kodu için **PEP 8** standartlarına uyun.
- Fonksiyon ve sınıflarınız için kısa **docstring** ekleyin.
- Değişken ve fonksiyon isimlendirmelerinde açıklayıcı İngilizce veya Türkçe isimler kullanın (proje genelindeki tutarlılığa dikkat edin).
- Gerekirse mevcut testleri çalıştırın veya yeni testler ekleyin.

## İletişim
Sorularınız için proje sahipleriyle iletişime geçebilirsiniz veya Discussions sekmesini kullanabilirsiniz.
