Dağıtım Rehberi

Agresif büyüme odaklı sistemin profesyonel ortamda kurulumu ve yönetimi.

Hızlı Kurulum

En güncel stabil sürümü yerel ortamınıza kurmak için aşağıdaki terminal komutlarını izleyin.

# 1. Projeyi İndirin
git clone https://github.com/alptigingorkem-coder/bist30_ai_trader.git
cd bist30_ai_trader

# 2. Sanal Ortam & Bağımlılıklar
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
